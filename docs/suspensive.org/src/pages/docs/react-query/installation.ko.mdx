# 설치하기

@suspensive/react-query는 npm에 있습니다. 최신 안정버전을 설치하기 위해 아래 커맨드를 실행하세요

[![npm version](https://img.shields.io/npm/v/@suspensive/react-query?color=000&labelColor=000&logo=npm&label=)](https://www.npmjs.com/package/@suspensive/react-query)

```shell npm2yarn
npm install @suspensive/react-query @tanstack/react-query@4
```

<Callout type='info'>

@suspensive/react-query는 내부적으로 peerDependencies로 @tanstack/react-query v4를 사용합니다. 때문에 package.json의 dependencies에 @tanstack/react-query의 올바른 버전이 설치되어있는지 확인하세요. 이유는 [어떤 이유로 사용하나요?](./motivation)에서 설명하고 있습니다.

</Callout>
