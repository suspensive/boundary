# SetAtom

<Callout type='experimental'>

`<SetAtom/>`는 실험 기능이므로 이 인터페이스는 변경될 수 있습니다.

</Callout>

import { Suspense } from '@suspensive/react'
import { UserProfile } from '~/components'
